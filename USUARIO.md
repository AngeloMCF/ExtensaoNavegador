# Manual do Usuário - Atalhos Pessoais

A extensão **Atalhos Pessoais** foi desenvolvida para centralizar e agilizar o acesso a links de ambientes de trabalho, ferramentas úteis e documentações técnicas diretamente no seu navegador.

## Funcionalidades Principais

A extensão organiza os links em categorias lógicas para facilitar o fluxo de trabalho diário:

*   **Painel de Ambientes**: Acesso rápido a URLs de Produção, Homologação e Desenvolvimento para diversos clientes e sistemas.
*   **Ferramentas Úteis**: Links diretos para formatadores de código (SQL, HTML), geradores de dados (CPF) e ferramentas de IA (ChatGPT, Manus, Claude).
*   **Wiki e Documentação**: Central de links para guias técnicos como Commits Semânticos, Codificação de URL e bibliotecas de ícones.
*   **Textos Rápidos**: Botões para copiar mensagens pré-definidas para o clipboard (ex: avisos de reunião ou almoço) com apenas um clique.
*   **Side Panel**: Suporte para abertura lateral no navegador, permitindo que os atalhos fiquem visíveis enquanto você navega em outros sites.

## Navegação e Páginas

A interface é dividida em seções acessíveis pelo menu superior:

1.  **Home**: Exibe os cards de ambientes (PROD, HML, DEV), Utilidades e Ferramentas.
2.  **Wiki**: Uma lista dedicada a links de referência técnica e documentação.
3.  **Pull Request**: Uma página opcional com links diretos para repositórios ou guias de revisão de código.
4.  **Configurações**: Acessível pelo ícone de engrenagem, permite personalizar a experiência.

## Personalização e Temas

A extensão oferece diversas opções de personalização:

*   **Modo Escuro/Claro**: Alterne entre temas visuais clicando no botão de tema no cabeçalho.
*   **Temas Sazonais**: A extensão possui temas automáticos e manuais para datas comemorativas:
    *   Natal (com efeito de neve)
    *   Halloween
    *   Páscoa
    *   Carnaval (com efeito de confete)
    *   Réveillon
*   **Gerenciamento de Cache**: Na página de configurações, você pode limpar as configurações salvas localmente caso precise resetar a extensão.

## Como Usar

1.  Clique no ícone da extensão na barra de ferramentas.
2.  Navegue entre as abas **Home** ou **Wiki**.
3.  Clique em qualquer link para abrir em uma nova aba.
4.  Para botões com ícone de cópia, o conteúdo será enviado para sua área de transferência automaticamente.
5.  Verifique o indicador de versão e status de conexão no topo para garantir que a extensão está atualizada.
