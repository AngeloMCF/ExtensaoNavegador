# Atalhos Pessoais - Extensão de Navegador

![Versão](https://img.shields.io/badge/versão-3.17.0-blue)
![Manifest](https://img.shields.io/badge/manifest-v3-green)

A **Atalhos Pessoais** é uma extensão para navegadores baseados em Chromium projetada para aumentar a produtividade de desenvolvedores e analistas, centralizando links essenciais, ferramentas de formatação e mensagens rápidas em um único lugar.

## 🚀 Funcionalidades

- **Central de Links**: Organização de URLs para ambientes de Produção, Homologação e Desenvolvimento.
- **Ferramentas Integradas**: Acesso rápido a formatadores SQL/HTML, geradores de CPF e ferramentas de IA.
- **Mensagens Rápidas**: Copie textos frequentes (avisos de reunião, almoço, etc.) com um clique.
- **Temas Dinâmicos**: Suporte a modo claro/escuro e temas sazonais automáticos (Natal, Halloween, Carnaval).
- **Side Panel**: Interface otimizada para uso na barra lateral do navegador.

## 📂 Estrutura do Projeto

- `src/`: Código-fonte da extensão (HTML, CSS, JS e Ativos).
- `python/`: Scripts de automação para build e processamento de imagens.
- `build/`: Pasta de saída para os pacotes de distribuição.
- `manifest.json`: Arquivo de configuração da extensão (V3).
